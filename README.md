# Игры разума (JavaScript)

«Игры разума» — набор из пяти консольных игр.

## Установка

```
$ sudo npm link 
``` 


## Игра: "Проверка на четность"

Игроку показывается случайное число. Ему нужно ответить **yes**, если число чётное, или **no** — если нечётное.

#### Запуск игры

```
$ brain-even
```
